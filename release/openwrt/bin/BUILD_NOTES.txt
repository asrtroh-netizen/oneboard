[info] pkg linux binaries cannot be built on Windows (pkg only builds for win here).
[info] Build pkg binaries on a Linux host or CI runner:
       cd onebord && npm install && npm run build:openwrt
[info] Universal fallback (all arches): onebord-openwrt-node.tar.gz + opkg install node

Expected pkg outputs on Linux:
  bin/onebord-openwrt-amd64   (node18-linux-x64)
  bin/onebord-openwrt-arm64   (node18-linux-arm64)
  bin/onebord-openwrt-armv7   (node18-linux-armv7)

mipsel: no official pkg target — use node tarball only.
